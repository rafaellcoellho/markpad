Webapp for markpad, a notepad in the web with markdown.

## Project setup

```
npm i
```

### Compiles and hot-reloads for development

```
npm run serve
```

### Compiles and minifies for production

```
npm run build
```

### Lints and fixes files

```
npm run lint
```
